# ClawGuard 0.1.0

First public release of ClawGuard, a local-first skill auditor for OpenClaw agents.

## Highlights

- Static scanning for local skills with scored findings and operator-readable reports
- Local daemon plus CLI workflow for `status`, `scan`, `report`, `audit`, `allow`, and `block`
- Behavioral detonation for locally installed skills through `clawguard detonate <slug>`
- Unified reporting that merges static and behavioral evidence
- Automatic quarantine for suspicious skills
- macOS `launchd` user-service install, status, and uninstall flow
- Packaged root `clawguard` artifact with tarball-install smoke coverage

## Included In This Release

- OpenClaw workspace discovery and watcher-backed local monitoring
- Static staged-download detection and baseline scoring
- Podman-first detonation runtime with Docker compatibility
- Preserved evidence artifacts for reports and detonation runs
- Release validation automation through `pnpm release:check`

## Verification Summary

This release was verified through:

- full `pnpm release:check`
- packaged tarball install smoke
- installed-binary daemon and CLI validation
- macOS service install/status/uninstall validation
- real OpenClaw-backed integration checks using the official published OpenClaw container image with a bind-mounted host workspace

## Known Caveats

- macOS is the primary supported platform for this release.
- Podman remains the default detonation runtime even though Docker compatibility is supported.
- VirusTotal and ClawHub enrich reports but do not determine safety on their own.
- Detonation reduces risk by exercising workflow behavior in a sandbox; it does not prove a skill is safe.
- Node may still emit the experimental SQLite warning on startup in some environments.

## Install

```bash
npm install -g clawguard
clawguard daemon
clawguard status
```

## Common Commands

```bash
clawguard scan /path/to/skill --detailed
clawguard report <slug> --detailed
clawguard detonate <slug> --detailed
clawguard audit
clawguard service install
clawguard service status
clawguard service uninstall
```

## Upgrade Notes

- The packaged release now includes the detonation sandbox assets needed by the installed `detonate` command.
- Watcher stale rescans after quarantine are handled as informational issues instead of crashing the daemon.
